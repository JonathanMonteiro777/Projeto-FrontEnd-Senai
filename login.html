Desenvolvimento da Página de Login

A página de login foi desenvolvida utilizando as seguintes tecnologias:

  • HTML: Estrutura básica da página, com a criação dos formulários de login e cadastro. Utilizando tags como form, label, input, entre outras, para estruturar os campos de entrada e interações básicas.
  • CSS: Estilização da página, utilizando o framework Bootstrap para a responsividade e um design moderno. Foram utilizadas classes como d-flex, justify-content-center, form-control e btn para layout e aparência. Também é possível personalizar o estilo adicional com um arquivo CSS externo (como estilo.css).
  • JavaScript: Interação com o usuário. As funções login() e cadastro() são responsáveis pela simulação de autenticação e registro de usuários, respectivamente. Elas são chamadas ao clicar nos botões "Login" e "Cadastrar", usando o atributo onclick. Não há integração com banco de dados neste projeto.
  • jQuery: Biblioteca JavaScript utilizada para manipular o DOM de forma mais simples e rápida. Está presente para controlar a exibição dos formulários de login/cadastro e o botão "Voltar ao Topo", usando seletores e eventos com facilidade.
  • Bootstrap: Utilizado via CDN, é o framework principal de estilização visual e responsividade. Com ele, garantimos que a interface se adapte a diferentes tamanhos de tela e mantenha uma aparência moderna e consistente.

Este projeto é uma introdução ao desenvolvimento web com foco em front-end, ideal para quem está começando a praticar HTML, CSS, JavaScript e jQuery, além de incorporar o uso de frameworks como Bootstrap.
